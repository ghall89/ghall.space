Uses

As both a professional web developer, an amatuer Mac and iPhone developer, as well as a hobbyist blogger, I have a variety of tools I use to get my work done.

Last updated: June 17, 2026

Hardware

My current workstation is an M2 MacBook Air, which I have connected to an LG UltraFine 27” display, with a NuPhy Air75 V2 keyboard (with classic Mac-style keycaps), and a Logitech M720 mouse.

MacBook Air M4

  • OS: macOS Tahoe
  • RAM: 24GB
  • Storage: 512GB SSD + external 1TB Samsung SSD

iPhone 14

  • OS: iOS 27 Beta
  • Storage: 128GB

Software

Music

  • Groove - Full disclaimer, this is my own app. It’s my primary music player on my iPhone.
  • Doppler - My primary music player on Mac. It doesn’t have all the features I’d like, but it works a lot better for my needs than the Music app.

Text Editors

  • Zed - My daily driver for web development. I like it because it’s fast, minimal, and just works.
  • Nova - I very much want this to be my daily driver for web development…
  • Xcode - Where I do all my iPhone and Mac dev.
  • BBEdit - My go-to app for making quick edits, and for writing.

Productivity

  • Zen Browser - Primary web browser.
  • Obsidian - Jotting down and organizing notes across my Mac, iPad, and iPhone.

Other Desktop Apps

  • WindowKeys - For assigning custom keyboard shortcuts to macOS’s built-in window management.
  • ForkLift - Finder alternative I mainly use for moving or copying files quickly.

CLI Tools

My terminal emulator of choice is Ghostty. These are some of the CLI tools I find useful:

  • Homebrew
  • Atuin
  • Zoxide
  • trash
